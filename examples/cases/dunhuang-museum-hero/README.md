# 敦煌市博物馆 Hero｜最终实现案例

这是一个非官方网站概念提案，用来保存 Product Composer 在文化品牌站、完整首屏、艺术字、原创图像世界、精细按钮和多路由差异化布局上的最终实现结果。

## 设计指纹

- 首页采用“暗色矿物洞壁包围明亮沙海”的空间入口，Hero 精确占满首个视口，飞天人物、标题、漆器按钮与三条内容路径共同完成第一屏叙事。
- 展览、馆藏、数字博物馆与参观页分别使用编辑索引、器物图谱、暗色空间地图和安静手卷路径，不复用同一套 Hero 构图。
- 宋体负责语义标题与正文，马善政毛笔字只承担单个艺术字；矿物青、朱砂、旧金、纸本与深墨各有固定角色。
- 交互保持直接：原生滚动、真实路由、移动菜单、馆藏筛选与详情、数字模式切换、参观偏好切换；没有滚动劫持、磁吸按钮、自定义光标或装饰性粒子。

## 本地运行

在本目录执行：

```bash
python3 serve.py
```

然后打开 `http://127.0.0.1:4173/`。服务脚本只使用 Python 标准库，并为以下前端路由提供刷新回退：

- `/`
- `/exhibitions`
- `/collection`
- `/digital`
- `/visit`
